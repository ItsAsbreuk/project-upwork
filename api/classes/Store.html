Subscribes a callback function to whenever Redux changes its state. Typically, this should be set within the "componentDidMount"-method of a React-Component.

The return-value is a function that should be used to detach the subscriber. Typically, detaching should be set within the "componentWillUnmount"-method of a React-Component.
